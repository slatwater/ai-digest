# 影幕滑入效果（Letterbox Intro）

**一句话概要**：用 2 个 fixed div + GSAP transform 归零，实现上下黑条同步滑入的电影感开场动效。

**适用场景**：首屏 intro 动画、展示型 / 作品集网站过场、需要电影感遮幕的任意 GSAP 时间轴。

---

## 核心要点

- `fixed` 定位锁住视口上下边缘，不随页面滚动
- `transform: translateY(±100%)` 把黑条藏到画外（不占布局、无闪烁）
- GSAP 把 transform 归零即完成滑入，逻辑极简
- `pointer-events: none` 防止黑条遮挡点击

---

## 三步模板（直接抄）

**Step 1 — HTML**
```html
<div class="letterbox top"></div>
<div class="letterbox bot"></div>
```

**Step 2 — CSS**
```css
.letterbox {
  position: fixed; left: 0; right: 0;
  height: 8vh;
  background: var(--ink);
  z-index: 100;
  pointer-events: none;
}
.letterbox.top { top: 0;    transform: translateY(-100%); }
.letterbox.bot { bottom: 0; transform: translateY(100%); }
```

**Step 3 — GSAP（接入已有时间轴 `intro`）**
```js
intro.to('.letterbox.top', { y: 0, duration: 0.6, ease: 'expo.out' }, 0)
     .to('.letterbox.bot', { y: 0, duration: 0.6, ease: 'expo.out' }, 0)
```

---

## 注意事项

- `z-index: 100` 需高于页面其他元素，否则黑条会被遮住
- `pointer-events: none` 必须加，否则黑条停留期间会吃掉点击事件
- 高度 `8vh` 可按视觉需要调整，常见范围 6–12vh
- 两条 `.to()` 的第三参数均为 `0`，确保上下同步启动
