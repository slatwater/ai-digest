## 核心发现

1. **文字规则有分层天花板**：
   - DESIGN.md（token）：文字规则 100% 有效
   - LAYOUT.md（版式/节奏/纪律）：文字规则 ~90% 有效
   - ASSETS.md 结构层（位置/密度/尺寸/白名单）：文字规则 ~70% 有效
   - **ASSETS.md 协调层（图与图的视觉一致性）：文字规则失效**

2. **每加一层规则会暴露下一层的缺失**：
   - A 平庸 → 暴露缺 LAYOUT
   - B 大众 → 暴露缺 Editorial Upgrade
   - C 乱 → 暴露缺 Hierarchy Discipline
   - C' 空 → 暴露缺 ASSETS
   - C'' 乱（新一种）→ 暴露**规则范式本身的天花板**

3. **提取工具能做的部分**：~30%（颜色/字体/间距/阴影/圆角等离散 token）

4. **必须靠人 + LLM 翻译的部分**：~70%（氛围、Don'ts、版式模式、图片气质、动效）

5. **规则文本无法跨越的最后一层**：图片之间的视觉协调 + 动效气质 —— 必须多模态或人工策展
