# CloakBrowser：源码级反指纹检测 Chromium 浏览器

## 要点

网站的反爬系统（比如 Cloudflare、reCAPTCHA）

CloakBrowser 的解法

不是在 JavaScript 层"打补丁"（那很容易被更底层的检测识破），而是直接改 Chromium 的 C++ 源码，打了 57 个补丁，让编译出来的浏览器二进制文件在这些底层指纹上和真实 Chrome 完全一致。

结果就是：

检测场景	结果

reCAPTCHA v3	得分 0.9（真人水平）

Cloudflare Turnstile	全部通过

30 个检测站点	30/30 通过

对开发者意味着什么

如果你已经有 Playwright / Puppeteer 项目，只改 3 行 import，业务代码一行不动，直接接入。本质上就是把"容易被认出的浏览器"换成"伪装得极好的浏览器"。
