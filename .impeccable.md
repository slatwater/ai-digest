## Design Context

### Users
技术研究者、AI/ML 从业者。使用场景：看到一条前沿 AI 资讯 → 丢链接进来 → 得到深度结构化分析 + 可运行 Demo。长时间沉浸式阅读，需要信息密度高但不疲劳。

### Brand Personality
权威、克制、精确 — 像一个你信赖的高级研究员递给你的简报。

### Aesthetic Direction
- **参考**: Vercel / Stripe — 黑白为骨，大留白，极致简洁，排版即设计
- **情绪**: 专业沉稳，让人信赖，不卖弄但有品质感
- **配色**: 浅色模式为主。中性色为基底，但需要一个克制的强调色打破灰调单调（考虑墨绿、深靛、琥珀等沉稳色调，而非科技蓝/紫）
- **反面参考**: 拒绝花哨渐变、霓虹色；拒绝蓝+白标准 SaaS 模板风；拒绝全灰无亮点
- **主题**: Light mode

### Design Principles
1. **排版即层级** — 用字号、字重、颜色对比建立信息层级，而非边框和卡片。标题大且重，辅助信息小且淡。
2. **克制的强调** — 一个颜色用在刀刃上（阶段指示、关键操作），比五个颜色到处撒更有力。
3. **呼吸感** — 充足的留白，内容不拥挤。密集信息区用间距节奏分组，而非线条分割。
4. **内容为王** — UI 退后，分析内容前进。界面是承载信息的透明容器，不抢注意力。
5. **精确的细节** — 对齐、间距、圆角保持高度一致。每个像素都有意图。
